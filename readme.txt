Описание правил работы WriterGuestsStatus
1. Первый объект из буфера должен начинаться на "TAp" (в противном случаи вызывает переподключение к Сокету)
2. В буфере порции по 208 байт.
    2.1 chTypePass: str  # 64 байта
    2.2 chObjName: str  # 64 байта
    2.3 chHolderName: str  # 64 байта
    2.4 dwCardNumber: int  # 4 байта
    2.5 # UnknownType: неизвестные данные  # 4 байта
    2.6 dtRealTimePass: double  # 8 байта  (преобразуется double(дельта времени от 1899.12.30) в datetime)
3. Таблица sac3.requeststatus в БД должна иметь поля с FID 5, 6
    где 5 это "Вход осуществлён"
    где 6 это "Выход осуществлён"
